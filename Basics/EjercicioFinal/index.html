Restaurante JS

Esta página tiene el fin de mostrar la información sobre el ejercicio a desarrollar y las instrucciones de uso para ejecutar la solución propuesta por mí.

Requerimientos del Ejercicio

Se requiere un sistema que pueda correr en consola que permita la atención al cliente en un restaurante. El sistema debe tener la capacidad de:

  • Mostrar el menú disponible.
  • Permitir que el usuario pida elementos del menú.
  • Mostrar el costo total que el usuario debe pagar.
  • Cobrar al usuario.
  • Reportar el monto total de ventas realizadas.

Instrucciones de Uso

  1. Abra la consola del navegador del navegador de su preferencia.
  2. Por defecto lo primero que verña será el menú, en caso de que no aparezca ejecute menu().
  3. Haga el pedido de los productos de su preferencia con la función order() dándole como parámetros arrays separados por coma con el formato ["idProducto", cantidad]. Si por ejemplo ejecutara order(["CR001", 2], ["B002", 1]), significaría que está pidiendo 2 hamburguesas sencillas y una gaseosa.
  4. Para obtener el total que debe pagar por el pedido, use la función orderDetail().
  5. Para pagar el pedido use la función payOrder("nombre", monto), donde "nombre" es el nombre del cliente que va a pagar y monto es el valor del dinero con el que va a pagar.
  6. Sí desea ver el historial de ventas use la función showOrderHistory().
  7. Por último, si desea obtener el monto total por todas las ventas realizadas use la función totalSales()
