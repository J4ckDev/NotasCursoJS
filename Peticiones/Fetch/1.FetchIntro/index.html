Javascript Asíncrono y Fetch

Fetch es la versión moderna para enviar, acceder y manipular peticiones HTTP. Este provee un método global 

fetch()
que proporciona de forma fácil y lógica para obtener los recursos de forma asíncrona por la red. Como está basado en promesas, contiene response y reject internos. Algunos de los métodos que tiene son los siguientes: 

  • arrayBuffer()
    : Archivos binarios en bruto (mp3, pdf, jpg, etc). Es utilizado cuando se requiere manipular el contenido de un archivo. 
  • blob()
    : Archivos binarios en bruto (mp3, pdf, jpg, etc). Es utilizado cuando no es necesario manipular el contenido y solo se va a trabajar con el archivo original. 
  • clone()
    : Crea un clon de un objeto de respuesta, idéntico en todos los sentidos, pero almacenado en una variable diferente. 
  • formData()
    : Es utilizado para leer los datos de un formulario. 
  • json()
    : Convierte los archivos JSON en un objeto javascript. 
  • text()
    : Es utilizado para leer un archivo de texto. Siempre codifica en UTF-8.
